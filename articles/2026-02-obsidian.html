社内ナレッジベース構築（Obsidian）

社内プロジェクト

2026-01-20 ~ 2026-02-05

社内の業務ナレッジをObsidianで構造化し、AIで検索・要約可能にする

Claude Cursor GPT
ナレッジ管理 社内ツール ドキュメント

自動化内容

  • 既存ドキュメント（Word/PDF）をMarkdownへ自動変換
  • Claude を使ったドキュメント要約の自動生成
  • Cursor でObsidianプラグインのカスタマイズスクリプトを作成
  • GPT API を使った社内Q&Aボットのプロトタイプ構築

成果

  • 約200件のドキュメントをMarkdown化し、Obsidianに集約
  • ナレッジ検索時間：平均15分 → 4分に短縮
  • 新人オンボーディング資料の自動生成に活用開始

再利用可能ポイント

  • Word/PDF → Markdown変換スクリプトは汎用的に利用可能
  • Obsidianのフォルダ構成テンプレートを作成済み
  • Q&Aボットのプロンプト設計パターンは他部署にも展開可能

課題・改善点

  • 図表を含むドキュメントの変換精度に課題あり
  • Obsidianの導入・運用に社内教育が必要
  • Q&Aボットの回答精度向上にはファインチューニングの検討が必要
