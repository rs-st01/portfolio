この記事について

ここまで紹介してきた事例以外にも、Stable Diffusion と Claude Code を日常的に活用しています。

現在はプライベートな作品が中心のため公開できるものが限られていますが、ポートフォリオ用の作品を別途制作し、今後このサイトで公開予定です。

この記事では、それぞれのツールで何ができるのかを先行して紹介します。

Stable Diffusion（画像生成AI）

何ができるか

Stable Diffusion は テキストの指示（プロンプト）から画像を生成できるAI です。写真風のリアルな画像からイラスト・アニメ調の作品まで、幅広いスタイルに対応しています。

  • イラスト制作 — キャラクターデザイン、背景イラスト、コンセプトアートなど
  • 写真風画像の生成 — 広告素材や商品イメージのたたき台として
  • 既存画像の加工 — img2img（画像から画像への変換）で、ラフスケッチを高品質な画像に仕上げるなど
  • LoRA・ControlNet — 特定の画風やポーズの制御も可能

ローカルPCで動作するため、クラウドサービスと違ってクレジット消費を気にせず何度でも生成を試せます（本サイトの他の事例で登場する nanobananaPro も Stable Diffusion がベースの技術です）。

コンテンツ規制への対策としても有効

Gemini などのクラウド型AI画像生成サービスは、安全性のためのコンテンツ規制が非常に厳しく設定されています。たとえば 子育て訴求用の子どもの写真素材 や、化粧品の肌表面の断面図解 といった、業務上まったく問題のない画像でも規制に引っかかって出力できないケースがあります。

Stable Diffusion はローカルで動作するため、こうした 過剰な規制に縛られずに業務用素材を生成できる というメリットもあり、クラウドサービスの規制対策として取り入れたいという意図もあります。

現在の活用

主にイラスト制作で活用しています。ポートフォリオとして公開できる作品を準備中です。

Claude Code（AIコーディングツール）

何ができるか

Claude Code は ターミナル上で動作する対話型のAIコーディングアシスタント です。自然言語で「こういうツールが欲しい」と伝えるだけで、設計からコーディング、ファイル作成、デバッグまで一括で行ってくれます。

  • 簡易Webツールの開発 — フォーム、計算ツール、データ変換ツールなど
  • スクリプトの作成 — データ処理、ファイル操作、自動化スクリプトなど
  • 既存コードの改修・リファクタリング

Cursor との違いは、IDE（統合開発環境）を使わずにターミナルだけで完結する点です。ちょっとしたツールを素早く作りたいときに特に威力を発揮します。

現在の活用

簡易Webツールなどを自分のプライベート用に作っていますが、ここで紹介してもいまいちなので、改めて業務の参考になりそうなものができれば紹介します。

今後の予定

Stable Diffusion・Claude Code それぞれのポートフォリオ作品を制作し、このサイトで事例として公開する予定です。公開時にはこの記事を更新、または個別の事例記事として追加します。
